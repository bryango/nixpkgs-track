# nixpkgs-track

Track pull requests across Nix channels.

## Installation

### Nix

```
nix run github:uncenter/nixpkgs-track
```

### Cargo

```sh
cargo install --git https://github.com/uncenter/nixpkgs-track.git
```

## Usage

```
nixpkgs-track <pull-request> [OPTIONS]
```
